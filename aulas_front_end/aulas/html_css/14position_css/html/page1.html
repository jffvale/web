Position não é uma tag HTML5, mas sim uma propriedade CSS que controla o posicionamento de elementos em uma página web.

Essa propriedade é usada para determinar como um elemento deve ser posicionado em relação aos seus elementos pai e vizinhos.

A propriedade position é frequentemente usada em conjunto com outras propriedades CSS, como top, right, bottom, left, para ajustar ainda mais o posicionamento do elemento na página.

Aqui estão alguns dos valores que a propriedade position pode ter:

  • static:

    • Este é o valor padrão.

    Elementos com position: static; são posicionados de acordo com o fluxo normal do documento, o que significa que eles serão exibidos na ordem em que aparecem no código HTML.


  • relative:

    • Elementos com position: relative; são posicionados em relação à sua posição original no layout normal.

    Se você usar propriedades como top, right, bottom ou left, o elemento será movido a partir de sua posição original.


  • absolute:

    • Elementos com position: absolute; são removidos do fluxo normal do documento e posicionados em relação ao elemento pai mais próximo que não tem position: static; (ou ao elemento <html> se nenhum elemento pai atender a esse critério).

    Se não houver um elemento pai que atenda a esses critérios, o elemento com position: absolute; será posicionado em relação ao elemento <html>.

    Este valor é frequentemente usado para criar layouts complexos.


  • fixed:

    • Elementos com position: fixed; são removidos do fluxo normal do documento e são posicionados em relação à janela de visualização (a área visível na tela), permanecendo fixos mesmo quando a página é rolada.

    Isso é comumente usado para criar barras de navegação que permanecem visíveis enquanto o usuário rola a página.

  • sticky:

    • Elementos com position: sticky; são um tipo híbrido entre relative e fixed.

    O elemento é tratado como relative até que ele atinja um determinado ponto durante o rolar da página, momento emque se torna fixed.
