Unidades de Medida para CSS

No HTML5, você pode especificar medidas usando diferentes unidades. Aqui estão algumas das unidades de medida mais comuns:

Pixels (px): Unidade de medida fixa. Um pixel é a menor unidade de medida em um dispositivo de exibição.

  • Exemplo: width: 200px;

    • Porcentagem (%): Relativa ao elemento pai. Por exemplo, se um elemento pai tiver uma largura de 400 pixels e um filho tiver width: 50%;, o filho terá uma largura de 200 pixels.

  • Exemplo: width: 50%;

    • Em (em): Relativa ao tamanho da fonte do elemento pai. Se o tamanho da fonte do elemento pai for 16 pixels e um filho tiver font-size: 1.5em;, o filho terá uma fonte de 24 pixels.

  • Exemplo: font-size: 1.5em;

    • Rem (rem): Relativa ao tamanho da fonte do elemento raiz (normalmente o ). Isso torna mais fácil manter a consistência em toda a página.

  • Exemplo: margin: 1rem;

    • Viewport Width (vw): Relativa à largura da viewport. 1vw é 1% da largura da viewport.

  • Exemplo: width: 50vw;

    • Viewport Height (vh): Relativa à altura da viewport. 1vh é 1% da altura da viewport.

  • Exemplo: height: 75vh;

    • Viewport Minimum (vmin): Relativa ao menor valor entre vw e vh.

  • Exemplo: font-size: 3vmin;

    • Viewport Maximum (vmax): Relativa ao maior valor entre vw e vh.

  • Exemplo: margin: 2vmax;

    • Estas são apenas algumas das unidades de medida disponíveis no HTML5 e CSS3. A escolha da unidade dependerá do contexto e dos requisitos específicos de design do seu projeto.
